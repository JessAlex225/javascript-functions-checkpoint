## Project Description

This project was completed as part of the JavaScript Foundations checkpoint.

- String manipulation
- Array manipulation

All functions were implemented and tested successfully using JavaScript and Node.js.

---

## Features Implemented

### String Manipulation Functions

#### 1. Reverse a String

Reverses a given string.

Example:

```javascript
inverserChaine("bonjour");
```

Output:

```javascript
"ruojnob"
```

---

#### 2. Count Characters

Counts the number of characters in a string.

Example:

```javascript
compterCaracteres("Bonjour");
```

Output:

```javascript
7
```

---

#### 3. Capitalize Words

Capitalizes the first letter of every word in a sentence.

Example:

```javascript
majusculeMots("bonjour tout le monde");
```

Output:

```javascript
"Bonjour Tout Le Monde"
```

---

### Array Functions

#### 4. Find Maximum Value

Returns the largest value in an array.

#### 5. Find Minimum Value

Returns the smallest value in an array.

#### 6. Sum of Array

Calculates the sum of all values in an array.

#### 7. Filter Array

Returns only the elements matching a given condition.

Example:

```javascript
[1, 2, 3, 4, 5, 6]
```

Output:

```javascript
[2, 4, 6]
```

---

### Mathematical Functions

#### 8. Factorial

Calculates the factorial of a number.

Example:

```javascript
factorielle(5);
```

Output:

```javascript
120
```

---

#### 9. Prime Number Check

Determines whether a number is prime.

Examples:

```javascript
estPremier(7);
```

Output:

```javascript
true
```

```javascript
estPremier(10);
```

Output:

```javascript
false
```

---

#### 10. Fibonacci Sequence

Generates the Fibonacci sequence for a specified number of terms.

Example:

```javascript
fibonacci(7);
```

Output:

```javascript
[0, 1, 1, 2, 3, 5, 8]
```
